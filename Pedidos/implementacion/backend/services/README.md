# Backend Services

Servicios de negocio previstos para el módulo [`Pedidos`](../../index.html).

## Casos de uso principales

- crear y actualizar pedidos
- gestionar rondas
- validar pagos y evidencias
- sincronizar cocina
- registrar cortesías
- emitir nota de crédito mock

## Regla arquitectónica

Los servicios deben recibir datos validados y devolver objetos de dominio limpios.
