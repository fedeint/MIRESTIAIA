1. selección de mesa
2. agregar productos
3. crear ronda
4. enviar a cocina
5. continuar con nuevas rondas

2. validación operativa
3. comprobante
4. pago
5. cierre de entrega

1. pedido creado o sincronizado
2. preparación
3. pago
4. comprobante
5. recojo final
